# Front Integration
### Plug-in to manage GitHub Issues

### For Development:

Start React watcher

    npm run start
    
Start server

    nodemon server.js

On connection start in terminal:

    ssh -f -N -T -R3123:localhost:3000 root@direct.meowsergirl.com
    ssh -f -N -T -R3124:localhost:3010 root@direct.meowsergirl.com
